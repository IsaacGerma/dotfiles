# Dotfiles

My dotfiles

# Programs I Use
* Distro - Arch Linux
* Shell - bash
* Terminal - Alacritty, sometimes others
* Text editor - neovim
* WM - i3
* Status Bar - i3blocks
* Launcher - Rofi
* File Manager - lf and pcmanfm
* Browser - Librewolf
* Video Player - mpv
* Music - mpd + ncmpcpp
* RSS feed reader - newsboat
* Scripts - some borrowed, some original

# Other things
* vim plugins with [vim-plug](https://github.com/junegunn/vim-plug)
